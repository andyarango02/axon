# Axon

Sistema de cotización inteligente por WhatsApp.

## Objetivo

Permitir que clientes soliciten cotizaciones mediante WhatsApp y que un sistema basado en IA:

1. Comprenda la necesidad del cliente.
2. Consulte productos y servicios disponibles.
3. Genere una pre-cotización automáticamente.
4. Solicite aprobación humana.
5. Envíe la cotización al cliente.
6. Realice seguimiento comercial de la oportunidad.

## Stack tecnológico

### Backend

* Node.js
* Express
* PostgreSQL (Supabase)

### Integraciones

* Twilio WhatsApp API
* OpenAI API

### Frontend

* React
